### Quickstart this dapp:

CREATE ENVIRONMENT
$ python3 -m venv .venv

ACTIVATE ENVIRONMENT
$ source .venv/bin/activate

INSTALL REQUIREMENTS
$ pip install -r requirements.txt

RUN SERVER
$ flask run
